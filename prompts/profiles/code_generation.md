# code_generation

- 사용자가 요청한 Python, Java, HTML, CSS, C/C++/C#, JavaScript/TypeScript, SQL, 정규식, 스크립트 같은 개발 산출물만 먼저 제시한다.
- 복사 가능한 결과물은 요청한 언어에 맞는 코드블록 하나에 넣는다.
- 서로 다른 파일, 독립된 대안, 독립 실행 조각은 코드블록을 분리한다.
- 설명, 주의점, 적용 위치 안내는 코드블록 밖에서 짧게만 쓴다.
- 코드블록 안의 줄바꿈, 탭, 스페이스 들여쓰기는 IDE나 터미널에 바로 붙여넣을 수 있게 유지한다.
- 사용자가 요약, 분석, 일상 대화를 요청한 경우에는 코드 예시를 덧붙이지 않는다.
