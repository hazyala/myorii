# 🗺️ Myorii Roadmap

Myorii의 장기적인 개발 방향과 버전별 목표를 정의한다.

---

# 프로젝트 목표

Myorii는 메뉴바 및 시스템 트레이에 상주하며,

사용자가 가장 자주 사용하는 생산성 기능을 창 이동 없이 즉시 사용할 수 있도록 돕는 데스크톱 컴패니언 애플리케이션을 목표로 한다.

---

# V1 - MVP

## 목표

핵심 사용자 경험 검증

---

## 현재 진행 상황

* PyQt6 기반 macOS 메뉴바 진입점 구현
* `assets/icons/menubar_icon.png`를 메뉴바 아이콘으로 연결
* 메뉴바 아이콘 클릭 시 메인 창 열기/닫기 토글
* 메뉴바 아이콘 위치를 기준으로 글래스 팝오버 창 표시
* 현재 작업 화면을 강제로 전환하지 않는 창 표시 방식 적용
* 채팅, 할일, 메모 탭과 탭별 컨텐츠 영역 전환 구현
* Ollama `qwen3-vl:4b` 기반 텍스트 채팅 스트리밍 구현
* 사용자/Assistant 메시지 버블과 Assistant Markdown 렌더링 구현
* 반응형 메시지 버블 폭과 코드블록 스타일링 구현
* 파일명/함수명/변수명/명령어 후보 자동 코드블록 렌더링 구현
* 코드 클릭 복사와 `복사됨` 토스트 구현
* 응답 중 빨간 발바닥 바운스 인디케이터 구현
* 설정 모델 목록 비동기 로딩으로 앱 시작 지연 방지
* 현재 세션 대화 히스토리 유지 구현
* 중복 실행 시 메뉴바 아이콘이 추가로 생기지 않도록 단일 실행 잠금 적용
* 인터넷 연결 상태에 따른 온라인/오프라인 Header 표시 구현
* 채팅 기록 버튼과 대화 기록 저장 스위치 UI 구현
* 추후 이미지/파일 업로드를 위한 입력창 내부 `+` 액션 UI 구현
* Header `×` 닫기 버튼 구현
* 채팅 입력 Enter/Shift+Enter 동작, 자동 줄바꿈, 채팅 영역 단독 스크롤 구현
* PyInstaller frozen 실행에서 로컬 `platform/` 패키지와 표준 라이브러리 `platform.py` 충돌 보정
* `LSUIElement`가 적용된 PyInstaller `.app` 번들 설정 추가
* SQLite 기반 로컬 데이터 저장 구조 구현 (chat / todo / memo)
* 할일 탭 로컬 CRUD 시작점 구현
* 할일 체크 완료 처리와 드래그 재정렬 구현
* PyInstaller 번들에서 할일 카드 hitbox가 어긋나지 않도록 카드 opacity effect 제거
* 다음 작업: 네이밍 기능, 이미지 첨부, 메모 UI, 할일 편집/삭제 UI

# 장기 비전

Myorii는 단순한 AI 채팅 앱이 아니다.

사용자가 가장 자주 사용하는 생산성 기능을 하나의 작은 메뉴바 친구 안에 담아,

작업 흐름을 끊지 않고 즉시 도움을 받을 수 있는 데스크톱 컴패니언을 목표로 한다.
