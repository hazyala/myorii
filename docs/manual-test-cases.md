# 수동 테스트 케이스

이 문서는 사용자가 직접 수행할 테스트 목록이다. AI가 수행한 테스트 결과를 기록하는 문서가 아니며, 기능별 구현 상태를 확인하고 이후 디버깅과 오류 수정을 추적하기 위한 기준 목록으로 사용한다.

## 향후 기능 테스트 작성 규칙

새 기능 테스트는 기능 영역별 섹션을 추가하고, `ID`, `테스트 항목`, `구현 상태`, `테스트 상태`, `확인 내용` 형식을 유지한다. 구현 상태는 `구현 완료`, `일부 구현`, `구현 예정` 중 하나를 우선 사용한다.
